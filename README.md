# Muhammad Irfan's Personal Portfolio

A modern, high-performance personal portfolio website showcasing graphic design and development work. Built with a premium glassmorphism aesthetic, interactive animations, and a fully responsive layout.

## ‚ú® Features

- **Premium Design**: Sleek glassmorphism UI with a dynamic Aurora background.
- **Interactive Navigation**: Gooey navigation effect for desktop and a responsive hamburger menu for mobile.
- **3D Profile Card**: Interactive profile card with persistent 3D tilt effects and hologram overlays.
- **Smooth Animations**: Tailored scroll-based reveal animations and staggered entrance effects powered by Framer Motion.
- **Skill Showcase**: Icon-grid layout with interactive tooltips for various design and development tools.
- **Portfolio & Blog**: Organized galleries for graphic design projects, web development work, and blog posts.
- **Interactive Dock**: Modern contact dock for quick access to social media and professional links.
- **Performance Optimized**: Fully responsive design with lazy-loaded images for faster load times.
